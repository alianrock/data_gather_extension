# Chrome Web Store 商店信息

## 扩展名称
网页信息收集助手

## 简短描述（132字符以内）
智能网页助手：一键收集页面信息、AI生成摘要、高清截图、生成分享卡片、管理书签收藏。支持OpenAI/Claude，隐私优先设计。

## 详细描述

### 功能强大的网页信息收集工具

网页信息收集助手是一款功能全面的Chrome扩展，帮助您高效地收集、整理和分享网页内容。无论是研究学习、内容创作还是日常浏览，都能大幅提升您的效率。

---

### 核心功能

**1. 智能信息收集**
• 一键提取网页标题、URL、描述、关键词
• 自动识别页面主要内容和结构
• 支持结构化数据（JSON-LD）解析
• 提取页面所有图片和链接

**2. AI智能摘要**
• 集成OpenAI GPT和Anthropic Claude大模型
• 实时流式生成，立即看到结果
• 多种摘要风格：极简、专业、随意、社交媒体
• 支持中文、繁体中文、英文等多语言

**3. 高清网页截图**
• 一键捕获整个可见区域
• 支持PNG和JPEG格式
• 可调节图片质量
• 自动包含在收集数据中

**4. 分享卡片生成**
• 将网页信息组合成精美卡片
• 多种设计模板可选
• 支持下载和复制
• 一键分享到社交媒体

**5. 社交媒体分享**
• 微信（生成二维码）
• 微博
• Twitter/X
• Facebook
• LinkedIn
• 一键复制链接

**6. 智能书签管理**
• 完整的网页收藏系统
• 智能分类（技术、学习、新闻、娱乐等）
• 支持自定义分类和标签
• 强大的搜索和过滤功能
• 网格和列表双视图模式

**7. 云端同步（可选）**
• 支持Turso数据库同步
• 跨设备访问书签
• 完全由您控制数据
• 可随时禁用

**8. 灵活的API对接**
• 将收集的数据发送到自定义API
• 支持POST和PUT方法
• Bearer Token认证
• 完整的数据结构化传输

---

### 隐私优先设计

我们高度重视您的隐私：
✓ API密钥仅存储在本地Chrome存储
✓ 不会后台自动收集任何数据
✓ 所有数据流向完全由您控制
✓ 不包含任何跟踪或分析代码
✓ 完全开源，代码可审查

### 使用场景

**研究人员**：快速收集和整理参考资料
**内容创作者**：获取素材并生成摘要
**产品经理**：收集竞品信息和用户反馈
**学生**：整理学习资料和笔记
**开发者**：保存技术文档和教程
**普通用户**：管理日常收藏和分享

---

### 快速开始

1. 安装扩展后点击图标打开设置
2. 配置您的AI API（OpenAI或Anthropic）
3. 访问任意网页，点击扩展图标
4. 点击"收集信息"开始使用
5. 享受AI自动生成的摘要！

---

### 支持的AI服务

• OpenAI（GPT-3.5、GPT-4、GPT-4o等）
• Anthropic Claude（所有版本）
