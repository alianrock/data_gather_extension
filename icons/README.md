# 图标文件说明

请在此目录下放置以下尺寸的PNG图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

## 图标设计建议

- 使用简洁的设计，能够在小尺寸下清晰识别
- 建议使用与网页信息收集相关的图标，如文档、数据、AI等元素
- 可以使用以下工具生成图标：
  - [Favicon Generator](https://favicon.io/)
  - [RealFaviconGenerator](https://realfavicongenerator.net/)
  - Figma、Sketch等设计工具

## 临时解决方案

如果暂时没有图标，可以：
1. 使用在线图标生成器快速创建
2. 或者从manifest.json中临时删除图标配置
3. Chrome会使用默认的扩展图标
